# NextLevel

Мобильное Android-приложение — образовательная платформа с элементами геймификации. Позволяет пользователям находить и проходить курсы, отслеживать прогресс обучения и получать награды через интерактивные механики.

## Возможности

### Авторизация
- Регистрация и вход по email/паролю
- OAuth-авторизация через Яндекс
- Восстановление пароля

### Обучение
- Каталог курсов с поиском и фильтрацией
- Детальная страница курса с рейтингом и прогрессом
- Раздел "Мои курсы" с отслеживанием прогресса
- Раздел задач

### Геймификация
- **Колесо фортуны** — ежедневный спин с наградами
- **Матрица судьбы** — интерактивный инструмент

### Прочее
- Внутренняя почта / уведомления (с skeleton-загрузкой)
- Профиль пользователя
- Онбординг для новых пользователей
- Экраны поддержки, условий использования и оферты

## Архитектура

Проект построен по принципам **Clean Architecture** с паттерном **MVI** (Model-View-Intent).

```
User Action → Event
    ↓
ViewModel.onEvent(event)
    ↓
Reducer → новый State
    ↓
StateFlow → UI recomposition
    ↓
Side-effects через Effect channel
```

**Слои:**
- **UI** — Jetpack Compose экраны и компоненты
- **Domain** — use-cases и бизнес-логика
- **Data** — репозитории, сетевые источники, локальное хранилище
